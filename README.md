# Brainfuck interpreter

Интерпретатор, поддерживающий следующие инструкции:

- `>` – увеличить указатель (чтобы он указывал на следующую ячейку справа),
- `<` – уменьшить указатель (чтобы он указывал на следующую ячейку слева),
- `+` – увеличить значение в текущей ячейке на единицу,
- `-` – уменьшить значение в текущей ячейке на единицу,
- `.` – вывести байт по указателю (напечатать значение из текущей ячейки),
- `,` – принять один байт из входных данных и сохранить его значение в текущей ячейке,
- `[` – если байт по указателю (то есть в текущей ячейке) равен нулю, то перейти вперед по тексту программы на символ, следующий за соответствующей `]` (с учетом вложенности скобок),
- `]` – если байт по указателе (то есть в текущей ячейке) ненулевой, то перейти назад по тесту программы на символ `[` (с учетом вложенности скобок).
